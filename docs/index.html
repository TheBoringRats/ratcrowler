üîß Installation

Get RatCrawler up and running quickly

Python Version

1. Install Dependencies

pip install requests beautifulsoup4 networkx schedule feedparser newspaper3k lxml_html_clean

2. Clone Repository

git clone https://github.com/TheBoringRats/ratcrowler.git
cd ratcrowler

3. Quick Start

python3 main.py

Rust Version

1. Install Rust

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2. Build Project

cd rust_version
cargo build --release

3. Run Crawler

./target/release/rat-crawler crawl https://example.com
