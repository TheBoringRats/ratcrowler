üîß Basic Configuration

Essential settings for getting started with RatCrawler

Python Configuration

Core Crawling Settings

delay Request delay between pages (seconds)
Default: 1.5, Recommended: 1.0-3.0
max_depth Maximum crawl depth
Default: 3, Range: 1-10
max_pages Maximum pages to crawl
Default: 100, Range: 10-10000
user_agent Custom user agent string
Default: Enhanced-Web-Crawler/1.0

Domain & Scope Settings

stay_on_domain Restrict crawling to seed domain
Default: True, Type: boolean
allowed_domains List of allowed domains
Default: [], Type: list
db_path Database file path
Default: website_crawler.db

Rust Configuration

CrawlConfig Structure

pub struct CrawlConfig {
    pub user_agent: String,        // Default: "RatCrawler/1.0"
    pub timeout_secs: u64,         // Default: 30
    pub max_redirects: usize,      // Default: 5
    pub max_depth: usize,          // Default: 3
    pub max_pages: usize,          // Default: 100
    pub delay_ms: u64,             // Default: 1000
    pub respect_robots_txt: bool,  // Default: true
}

BacklinkConfig Structure

pub struct BacklinkConfig {
    pub user_agent: String,        // Default: "RatCrawler-Backlinks/1.0"
    pub timeout_secs: u64,         // Default: 60
    pub max_redirects: usize,      // Default: 5
}

üìù Complete Configuration Example

# Python Configuration Dictionary
config = {
    # Basic crawling settings
    'delay': 1.5,                    # Seconds between requests
    'max_depth': 3,                  # Maximum link depth
    'max_pages': 100,                # Maximum pages to crawl
    'user_agent': 'MyCrawler/1.0',   # Custom user agent

    # Domain restrictions
    'stay_on_domain': True,          # Stay within seed domain
    'allowed_domains': [],           # Additional allowed domains

    # Database settings
    'db_path': 'crawler.db',         # Database file path
    'recrawl_days': 7,               # Days between recrawls

    # Analysis settings
    'analyze_backlinks': True,       # Enable backlink analysis

    # Export settings
    'export_json': True,             # Export results as JSON
    'export_csv': False,             # Export data as CSV
}

üî¨ Advanced Configuration

Fine-tune RatCrawler for specific use cases

‚ö° Performance Tuning

Request Optimization

  • delay: Balance speed vs politeness (1.0-5.0s)
  • timeout: Request timeout (10-60s)
  • max_redirects: Maximum redirects to follow (3-10)
  • concurrent_requests: Parallel requests (1-10)

Memory Management

  • max_pages: Limit memory usage (100-10000)
  • content_size_limit: Max content size (1MB-10MB)
  • cache_size: URL cache size (1000-10000)

Database Optimization

  • batch_size: Database batch operations (10-100)
  • vacuum_interval: Database cleanup frequency
  • index_strategy: Database indexing approach

üéØ Content Filtering

URL Patterns

  • include_patterns: URL regex patterns to include
  • exclude_patterns: URL regex patterns to exclude
  • file_extensions: Allowed file extensions
  • path_filters: URL path filters

Content Types

  • allowed_mime_types: MIME types to crawl
  • min_content_length: Minimum content length
  • max_content_length: Maximum content length
  • language_filter: Content language filter

Quality Filters

  • min_word_count: Minimum word count
  • duplicate_threshold: Content similarity threshold
  • spam_score_threshold: Spam detection threshold

üöÄ Production Configuration

# Production-ready configuration
production_config = {
    # Performance optimized
    'delay': 2.0,
    'max_pages': 10000,
    'timeout': 30,
    'max_redirects': 5,

    # Content filtering
    'allowed_domains': ['example.com', 'blog.example.com'],
    'exclude_patterns': [
        r'\.pdf$',
        r'\.jpg$',
        r'/admin/',
        r'/private/'
    ],

    # Quality control
    'min_word_count': 50,
    'min_content_length': 500,
    'duplicate_threshold': 0.85,

    # Database optimization
    'db_path': '/data/crawler/production.db',
    'batch_size': 50,
    'vacuum_interval': 1000,

    # Monitoring
    'enable_metrics': True,
    'log_level': 'INFO',
    'stats_interval': 100,

    # Export & backup
    'export_json': True,
    'export_csv': False,
    'backup_interval': 3600,  # 1 hour
}

üåç Environment Variables

Configure RatCrawler using environment variables

Python Environment

MONGODB_URI Optional

MongoDB connection string for additional storage

MAX_CRAWL_DEPTH Default: 3

Maximum depth for crawling

MAX_PAGES_PER_DOMAIN Default: 100

Maximum pages to crawl per domain

USER_AGENT Optional

Custom user agent string

Rust Environment

RUST_LOG Optional

Set logging level (debug, info, warn, error)

DATABASE_URL Optional

Database connection URL

MAX_WORKERS Default: 4

Maximum number of worker threads

CACHE_SIZE Default: 10000

URL cache size in memory

üìÑ Environment File (.env)

# RatCrawler Environment Configuration
# Copy this to .env file in your project root

# Database Configuration
MONGODB_URI=mongodb://localhost:27017/ratcrawler
DATABASE_URL=sqlite:///ratcrawler.db

# Crawling Limits
MAX_CRAWL_DEPTH=3
MAX_PAGES_PER_DOMAIN=100
MAX_WORKERS=4

# Performance Settings
REQUEST_TIMEOUT=30
CRAWL_DELAY=1.5
CACHE_SIZE=10000

# User Agent
USER_AGENT=RatCrawler/1.0 (+https://github.com/yourusername/ratcrawler)

# Logging
RUST_LOG=info
LOG_LEVEL=INFO

# Export Settings
EXPORT_JSON=true
EXPORT_CSV=false
BACKUP_INTERVAL=3600

# Security
ALLOWED_DOMAINS=example.com,blog.example.com
BLOCKED_DOMAINS=spam.com,bad.com

# API Keys (if using external services)
GOOGLE_TRENDS_API_KEY=your_api_key_here
NEWS_API_KEY=your_api_key_here
